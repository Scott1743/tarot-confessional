# 塔罗树洞东方神秘视觉重设计

- 日期：2026-07-12
- 状态：实现中
- 范围：`assets/draw.html`、`assets/reading.html` 与原型牌组
- 设计参数：`VARIANCE 8 / MOTION 8 / DENSITY 3`

## 1. 重设计原因

首版黑色版画视觉具有较强的欧洲神秘学和古典塔罗气质，与中文用户所期待的东方神秘感不符。新版保留“高品质、非廉价占卜网站”的目标，但把视觉母题改为东方夜观、紫气、绢本矿物色、漆器和云气流动。

## 2. 设计命题

两张页面共同构成一次“夜观问心”：

1. 抽牌页是夜间观象台。用户进入后先看到流动紫气和缓慢旋转的星轨，牌堆位于天地轴线中心。
2. 洗牌时星轨收束，背景光影加速，牌背像从云层中展开。
3. 翻牌后光从牌面扩散到三个牌位，动作由未知转为可读。
4. 报告页是一卷在紫色夜幕中展开的绢本长卷，牌面、文字和行动建议沿纵向气脉排列。

## 3. 文化语言

采用：

- 宋画式细线与留白。
- 敦煌壁画启发的矿物紫、石青、朱砂和金色颗粒。
- 汉唐云气纹的连续曲线。
- 古代星图、浑天仪和二十八宿的结构感，但不直接复制文物图样。
- 紫檀、黑漆、绢本和金箔的材质层次。

避免：

- 直接使用具体宗教神像作为塔罗人物替代。
- 无来源地混用日本浮世绘、藏传法器和中国宫廷纹样。
- 紫色科技网格、霓虹外发光、发光圆球、通用玻璃卡片。
- 以“东方”名义堆叠龙、凤、灯笼、铜钱等装饰符号。

## 4. 色彩系统

| Token | 色值 | 用途 |
|---|---|---|
| `night` | `#12091D` | 最深天幕 |
| `ink-purple` | `#21102F` | 漆器与页面底色 |
| `imperial-purple` | `#54266F` | 云气主体 |
| `glow-purple` | `#A65BD0` | 动态光影与焦点 |
| `mineral-blue` | `#416A86` | 冷色平衡 |
| `cinnabar` | `#A94B52` | 少量叙事强调 |
| `gilded` | `#D6B36A` | 结构线、牌边与按钮 |
| `moon-white` | `#EEE8F2` | 主文字 |

紫色是主色，但通过蓝、朱砂、金与墨色建立多色层次，避免一屏只有同一色相。

## 5. 图像资产

### 5.1 牌面

媒介统一为“绢本设色与矿物颜料数字绘画”：细线轮廓、平涂矿物色、金色颗粒、局部漆黑留白。人物服装参考宽袍、披帛和古代旅行装束，但保持原创和时代模糊，不复制具体朝代礼服。

8 张牌重新解释为东方场景：

- 愚者：云游少年走向山门之外，白犬相随。
- 魔术师：术者在漆案上布置笔、剑、盏与玉环，天地方向贯通。
- 女祭司：司夜者静坐于双阙与帷幕之间，水镜在前。
- 皇后：园中主人管理桑、谷、果与流水，丰饶来自培育。
- 隐者：提灯者独行雪岭，远处寺观只作地景轮廓。
- 星星：人物在莲池边倾注双壶，东方星图在天空展开。
- 月亮：月门、双阙、黑白犬和水中曲径，暧昧但不恐怖。
- 太阳：孩童穿过朱门与花木庭院，金色日轮照亮生命力。

### 5.2 牌背

使用上下对称的浑天仪结构、双鱼、云气和星宿节点。不得生成可读文字、卦名、符咒或宗教印记。

### 5.3 背景

- `eastern-sky.jpg`：宽幅紫色夜空和横向云气，供抽牌页动态平移与缩放。
- `purple-silk.jpg`：深紫绢本纤维，供报告页和页面噪点层。
- 背景位图只提供真实材质，动画由 CSS 与 Canvas 完成。

## 6. 抽牌页动效

| 层 | 动效 | 意义 |
|---|---|---|
| 天幕底图 | 36 秒缓慢水平漂移和轻微缩放 | 表达时间流动 |
| 紫气 | 两层非圆形大面积云带交错移动 | 建立深度与呼吸感 |
| 星轨 Canvas | 低密度星点沿椭圆缓慢运动 | 暗示观象与秩序 |
| 牌桌光 | 指针移动产生克制视差 | 让页面回应用户在场 |
| 洗牌 | 星轨收束、牌堆分合、云气短暂加速 | 表达随机过程发生 |
| 选牌 | 牌背上浮并留下短暂金色余影 | 确认选择 |
| 翻牌 | 3D 翻转后牌位光脉依次点亮 | 从未知进入解读 |

所有动画只修改 `transform` 与 `opacity`。Canvas 使用固定帧率和设备像素比上限，页面隐藏时停止。`prefers-reduced-motion` 下禁用 Canvas 循环、背景漂移和复杂过渡。

## 7. 报告页动效

- 首次进入时封面题目、牌阵与正文依次显现。
- 三张牌在进入视口时轻微上移并恢复清晰度。
- 一条细金色气脉贯穿逐牌解读，只随滚动显现，不劫持滚动。
- 背景紫气比抽牌页更慢，避免干扰阅读。
- 打印模式完全静态，转为浅色宣纸底和深色文字。

## 8. 布局调整

### 抽牌页

- 移除西式档案编号感较强的等宽元数据布局。
- 标题改为纵向偏轴构图，牌阵名称以小型印章式文本块承载，但不模拟真实篆刻。
- 三个牌位以弧形气脉连接，不使用三个相同卡片容器。

### 报告页

- 从浅灰档案纸改为深紫长卷。
- 封面使用大面积动态背景和垂直题名。
- 正文使用月白文字与金色结构线，局部使用浅紫半透明绢本层，不使用玻璃卡片。
- 打印样式保留浅色纸张，确保可用性。

## 9. 验收

- 第一眼应能识别东方绢本和夜观气质，而不是欧洲版画。
- 紫色绚丽但正文对比度达到 WCAG AA。
- 动态背景始终存在，但不导致文字难读、移动端发热或眩晕。
- 洗牌、选牌、翻牌三个时刻有明显而不同的编排。
- 390 x 844 下无重叠，主按钮与三张牌可访问。
- 减少动作模式下功能完整且背景静止。
- 页面不加载任何远程资源。

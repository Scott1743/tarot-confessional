一 · 玩法

三张牌，三个视角，
三分钟看见自己

不需要懂塔罗，不需要记牌意。把问题放心底，让浏览器替你完成洗牌与选牌，然后把抽牌码交给我——剩下的，由我们一起慢慢说。

1

把问题放在心里

也许是「我要不要换工作」，也许是「我们之间到底怎么了」。不需要打字说出来。

2

点一下，让牌桌帮你开始

浏览器用加密随机洗一副完整的 78 张牌，从 22 张大阿卡纳到 56 张小阿卡纳，一张不多一张不少。

3

凭直觉选三张

牌面朝下，凭第一感觉挑出三张。系统会给你三种牌阵：现状 / 阻力 / 方向，或者自己 / 对方 / 关系。

4

复制抽牌码，回到对话

一段不带任何隐私信息的 TC1 编码，把牌面"安全地"交给我。我会用温柔的语言，把它说成你听得懂的故事。

二 · 私人阅读档案

你带走的，不只是一串抽牌码

每一次抽完，我会生成一份可以保存、打印、反复回看的私人阅读档案。温柔的封面、纵向气脉串联的牌位解读、最后留两件可行的事和三个留给自己的问题。

它会包含

  • 封面：日期、牌阵、你的问题（自动去隐私）
  • 牌阵全景：图像 + 牌位 + 正逆位
  • 逐牌解读：用你听得懂的语言，不堆术语
  • 综合判断：牌与牌之间在说什么
  • 可行之事：最多两项低风险下一步
  • 留待自问：最多三个能带走的反思
  • 边界说明：什么时候该去找专业帮助
"塔罗作为整理问题的象征工具，
不是对结果的保证。"

每一份报告顶部都印着这句话。我们不预测命运，只陪你把心事说清楚。

三 · 今晚就开始

装好 Skill，
说一句话就够了

塔罗树洞是一个 Agent Skill。你不需要跑任何脚本，不需要懂命令行——安装之后，像平时聊天一样跟 Agent 说话，它会替你处理一切。

① 一键安装

用一行命令装进你的 Agent：

npx skills add Scott1743/tarot-confessional

兼容 Claude Code、Codex、Cursor、GitHub Copilot 等所有主流 Agent。

② 对话提示词

安装完成后，复制下面这段话贴进对话——剩下的全部交给 Agent：

我想抽一次塔罗。
（可选：我最近在困扰 [你的事]）
请帮我开始。

Agent 会自动启动抽牌页面、等你抽完后解码牌面、生成解读报告。你只需要对话和点链接。

准备好开始了吗？

今晚，给自己十分钟的不被打扰。

npx skills add Scott1743/tarot-confessional

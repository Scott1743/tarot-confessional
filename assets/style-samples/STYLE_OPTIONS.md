# Full Deck Style Options

本目录用于确定完整 78 张牌的最终视觉风格。每个方向先生成“魔术师”和“月亮”两张，因为这两张同时覆盖人物、器物、建筑、动物、夜景与光影。

本轮实际使用项目此前已配置的 MiniMax 生图通道生成。最终候选位于各方向的 `final/` 目录，原始生成文件不进入 Git。

## A. 敦煌霓彩

东方绢本和敦煌矿物色的当代高饱和版本。主色使用玫红、石青、孔雀绿、珊瑚橙和柠檬黄，金线只负责结构。画面华丽、柔和、叙事性强，最接近当前 HTML 的紫色云气氛围。

## B. 漆彩撞色

以黑漆和金线为底，加入酸性绿、电光蓝、桃红与朱砂，像当代漆画和丝网版画。对比最强，缩小到手机卡牌尺寸时仍有辨识度，个性也最鲜明。

## C. 新工笔幻色

工笔线描结合现代编辑插画，大色块使用紫罗兰、青蓝、蜜橘、嫩黄和玉色。人物更精致，画面更轻盈，适合完整牌组保持统一，但神秘感相对克制。

所有方向共同要求：东方文化语言一致、原创构图、无文字和伪字、无 Logo、水印、符咒、具体宗教神像、西式中世纪服装或直接复制 RWS 构图。
